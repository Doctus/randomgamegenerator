Overview

The map editor provides tools for drawing and editing tileset-based maps. You must create a map with a valid tileset to use the map editor.

Draw Modes

The map editor has three distinct draw modes and a no-draw mode. The latter mode disables most map editor functions, allowing the widget to be left open when you're not drawing without interfering with other activities.

Single Tile Drawing

With a tile from the tileset selected, left-click on the map to draw that tile. Left-clicking and dragging will draw on all tiles over which the cursor is dragged.

Rectangle Drawing

With a tile from the tileset selected, left-click on the map and drag to another point to fill the rectangle bounded by those points with the selected tile.

Hollow Rectangle Drawing

This mode works exactly like rectangle drawing, but changes only the borders of the rectangle.

Additional Functions

Tile Selection

Select a tileset by clicking on a map to identify the map you wish to edit, then left-click on the tileset within the map editor to choose a tile.

It is also possible to "pick up" a tile on the map by ctrl-right clicking it.

Copying and Pasting

Shift-right click and drag to copy a rectangular area of tiles. Shift-left clicking in single tile mode will paste one full copy of the area, while shift-left clicking in rectangle drawing mode will fill the area with the copied tiles as a pattern.

Undo and Redo

It is possible to undo and redo most map editor drawing actions with no depth limit.

You can only undo and redo your own local changes, not changes made by other users. An undo action that affects a tile changed by another user will, however, overwrite that user's change when reverting to the old state.

It is not currently possible to undo paste actions.
